Inside Frontend 用のサンプルコードです。リファクタが間に合っておらずゴミが混じっており、ご容赦ください、、、

## 環境

```
$ node -v
v10.15.3

$ yarn -v
1.15.2
```

## 起動

```sh
$ yarn install

// mock server起動
$ yarn run mock:local

// localで起動
$ yarn run start:local

// prd用にbuild
$ yarn run build:prd
```
