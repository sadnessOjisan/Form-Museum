# Form Museum

Inside Frontend 用のサンプルコードです。

## 環境

```
$ node -v
v10.15.3

$ yarn -v
1.15.2
```

## 起動

```sh
$ yarn install

// localで起動
$ yarn run start:local

// prd用にbuild
$ yarn run build:prd
```
